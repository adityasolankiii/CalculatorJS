# Calculator

A simple calculator built using HTML, CSS, and JavaScript. This project provides basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear/reset button
- User-friendly interface
